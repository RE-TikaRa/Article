# 关于
这个页面的出现，主要是因为在某些不太理想的情况下，原博客的样式会出现一些不可控的问题，比如排版崩坏、样式丢失，或者看起来像是被某种未知力量干扰了一样（沉默）。为了避免内容在这种时候直接“不可阅读”，于是就有了这个类似备用存档的页面。

这里的文章内容和博客里的内容是一样的，没有隐藏剧情，也没有多周目分支（大概）。你可以把它理解为一个以“至少能正常看字”为目标的安全模式页面，用来在博客本体不太稳定的时候，继续维持信息输出的基本功能。

在实现方式上，这个页面由 Typora 编写，并使用 Vlook 主题进行渲染。相比完整博客系统，这种组合更加直接，也更不容易出现奇怪的渲染行为，适合用来做这种“以稳定为第一优先级”的存在（点头）。

总之，这里并不是博客的替代品，更像是一个安静待机的副本。当主站状态良好时，它可以被忽略；当主站出现异常时，它会顶上来，保证内容至少还能被读到。
